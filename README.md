## Pokretanje projekta

1. Kloniraj repo
2. Otvori u Visual Studiju
3. Pokreni migracije: View → Terminal u Visual Studiju, u pravom folderu (onaj gde je ITFirma.csproj):
   dotnet ef database update
4. Pokreni projekat (F5)
